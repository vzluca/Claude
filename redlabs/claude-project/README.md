# Proyecto "RedLabs" para claude.ai (Claude normal)

Cómo armarlo, paso a paso:

1. En claude.ai → **Proyectos** → **Crear proyecto** → nombre: **RedLabs**.
2. En **Instrucciones del proyecto**, pegá el contenido completo de `instrucciones.md`.
3. En **Conocimiento del proyecto**, subí estos archivos (están en este repo):
   - `redlabs/brand/REDLABS-manual-de-marca.pdf` — identidad visual y tono de voz
   - `redlabs/knowledge/datos/Centro_de_INFO_RedLabs_v2.xlsx` — catálogo, precios y stack (fuente de verdad)
   - `redlabs/knowledge/empresa.md`
   - `redlabs/knowledge/servicios.md`
   - `redlabs/knowledge/stack.md`
   - `redlabs/knowledge/clientes.md`
4. Listo: cualquier chat dentro del proyecto arranca sabiendo todo de RedLabs.

> Cuando actualices el Centro de INFO en Drive, reemplazá el xlsx del proyecto
> para que Claude siempre cotice con los precios vigentes.
