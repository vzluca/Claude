# RedLabs — Empresa

> Fuentes: "Centro_de_INFO_RedLabs_v2" (Drive, v2.0 al 10/07/2026), Brand Book v1.1 y sitio web.

## Qué es RedLabs

- **Misión**: empresa de automatización con IA. Conecta WhatsApp, Gmail, Drive, Sheets, Calendar
  y modelos de lenguaje con n8n como motor para que los negocios "corran solos". También diseña
  y programa sitios web con Claude Code (hosting en Firebase).
- **Servicios**: 14 servicios predefinidos de automatización (11 generales + 3 nuevos + 1 vertical
  inmobiliaria), 7 combos con descuento, automatizaciones personalizadas por niveles, y servicios
  web (landings y sitios con base de datos). Detalle en `servicios.md`.
- **Público objetivo**: negocios chicos y profesionales independientes (inmobiliarias como vertical
  original, contadores, abogados, coaches, dueños de comercios) que pierden tiempo en tareas
  repetitivas. Personas sin conocimientos técnicos.
- **Diferencial**: flujos ya construidos y probados que se implementan en días, no en meses;
  la parte complicada queda del lado de RedLabs; herramientas que el cliente ya usa.

## Modelo de negocio

| Modalidad | Cómo funciona |
|---|---|
| 🔵 **Alquiler** | Implementación reducida + cuota mensual. Si el cliente no paga, el flujo se desactiva. RedLabs mantiene el `.json`. Genera MRR. |
| 🟢 **Compra** | Precio único mayor. Si el cliente cancela el mantenimiento, recibe el `.json` y se va. Genera ticket alto. |
| 🟡 **Personalizado** | Precio base por nivel: Simple (hasta 8 nodos, u$s 200), Medio (hasta 18, u$s 450), Complejo (hasta 30, u$s 800) + u$s 15 por nodo extra. Más de 30 nodos → cotización manual. |
| 🌐 **Servicios web** | Pago único: landing u$s 200–250, sitio con base de datos u$s 450–550. Mantenimiento y hosting Firebase se cotizan aparte. |

Rango de precios de servicios predefinidos: desde u$s 110 impl. + u$s 20/mes (alquiler)
hasta u$s 650 único (ACM inmobiliario, compra).
